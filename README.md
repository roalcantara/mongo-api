## Install

`git clone https://github.com/roalcantara/mongo-api`

### Dependencies

- [git][5]
- [ASDF][6]
- [Direnv][7]
- [pre-commit][8]
- [gitlint][9]

## Usage

- Run `npm install` to install dependencies
- Run `npm start` to start the server
- Run `npm run dev` to start the server in development mode
